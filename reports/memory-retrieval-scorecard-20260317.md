## 结论
1. **长期规则 / 稳定事实**：表现稳定，1-11 中除“当前主记忆插件”外基本都是 Top1 精确命中。
2. **短期 / 会话型问题**：13、15 仍明显偏弱，说明当前 Layer 2 对“刚才决定了什么”这类问题还不够稳。
3. **精确数值型问题**：12 暴露出对固定阈值（`40k tokens`）的召回仍有盲点。
4. **L3 fallback 现状**：今天 benchmark 中有 5 条查询触发 Layer 3，但没有一条形成有效补充，其中 3 条出现 NotebookLM session lock / gateway timeout。
5. **当前判断**：Layer 2 已足够承担日常主链路；下一步优先级不是再折腾 builtin memorySearch，而是单独排查 Layer 3 fallback 可用性，并继续观察短期记忆召回。
